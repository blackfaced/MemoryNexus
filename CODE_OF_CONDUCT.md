# 📜 行为准则

## 我们的承诺

为了营造一个开放、友好的社区环境，我们承诺让所有参与者都有一个无骚扰的体验，无论年龄、体型、残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向。

## 我们的标准

### 鼓励的行为

- 🗣️ 使用欢迎和包容的语言
- 🙏 尊重不同的观点和经历
- 💬 优雅地接受建设性批评
- 🤝 关注社区共同利益
- ❤️ 向其他社区成员表达善意

### 不可接受的行为

- 🚫 使用带有性别歧视、种族主义或排他性的语言
- 🚫 人身攻击、贬低他人
- 🚫 公开或私下骚扰
- 🚫 未经同意发布他人私人信息
- 🚫 其他不道德、不专业的行为

## 执行

社区负责人有责任解释和执行本行为准则。对于不当行为的举报，可以联系项目维护者。

## 处理方式

违反本准则的行为将通过以下方式处理：

1. **口头警告**：轻微违规
2. **书面警告**：重复或较严重的违规
3. **暂时禁止**：严重违规
4. **永久禁止**：严重或持续违规

## 来源

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)。
