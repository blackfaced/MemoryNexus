# 📅 MemoryNexus 开发路线图

> 让记忆连接，让知识生长

---

## 版本规划

### 🚧 v0.1.0 - MVP (目标: 2-3个月)
> 最小可行产品，验证核心功能

**状态**: 🚧 进行中

| 功能 | 描述 | 优先级 |
|------|------|--------|
| ✅ 用户注册/登录 | JWT认证 | P0 |
| ✅ 基础记忆 CRUD | 创建、读取、更新、删除记忆 | P0 |
| ✅ 文本记忆存储 | 文字笔记存储和检索 | P0 |
| ✅ 向量检索 | 基于语义的内容检索 | P0 |
| ⬜ 图片上传 | 支持图片存储和展示 | P1 |
| ⬜ 基础标签 | 给记忆打标签分类 | P1 |
| ⬜ 搜索功能 | 全文+向量混合搜索 | P1 |

---

### 📦 v0.2.0 - 多模态 (目标: 3-4个月)
> 支持更多类型，扩展使用场景

| 功能 | 描述 | 优先级 |
|------|------|--------|
| ⬜ 语音转文字 | Whisper 集成 | P1 |
| ⬜ 图片向量 | CLIP 图片理解 | P1 |
| ⬜ 跨模态检索 | 用文字搜图片 | P2 |
| ⬜ 家庭成员 | 多用户/家庭支持 | P1 |
| ⬜ 角色区分 | 孩子/父母模式 | P2 |

---

### 🤖 v0.3.0 - AI 增强 (目标: 4-5个月)
> 加入主动智能助理

| 功能 | 描述 | 优先级 |
|------|------|--------|
| ⬜ AI 摘要 | 自动生成记忆摘要 | P1 |
| ⬜ 定时回顾 | 每日/每周自动回顾 | P1 |
| ⬜ 洞察发现 | 发现记忆中的关联 | P2 |
| ⬜ TODO 生成 | AI 辅助生成待办 | P2 |
| ⬜ 定期报告 | 周报/月报生成 | P2 |

---

### ☁️ v0.4.0 - 云端化 (目标: 5-6个月)
> 托管服务，支持多云部署

| 功能 | 描述 | 优先级 |
|------|------|--------|
| ⬜ SaaS 部署 | 一键云端部署 | P1 |
| ⬜ 火山引擎适配 | 字节云部署指南 | P2 |
| ⬜ 阿里云适配 | 国内云部署指南 | P2 |
| ⬜ AWS/GCP 适配 | 海外云部署指南 | P2 |
| ⬜ 监控告警 | 生产环境监控 | P2 |

---

### 💎 v1.0.0 - 正式版 (目标: 6-8个月)
> 功能完善，稳定可靠

| 功能 | 描述 | 优先级 |
|------|------|--------|
| ⬜ 主动提醒 | 基于记忆的智能提醒 | P1 |
| ⬜ API 开放 | 外部 Agent 接入 | P1 |
| ⬜ 插件系统 | 扩展机制 | P2 |
| ⬜ 性能优化 | 大规模数据优化 | P1 |
| ⬜ 安全审计 | 安全加固 | P1 |
