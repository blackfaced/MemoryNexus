## 核心模块

### 1. 用户与家庭

```
┌─────────────┐     ┌─────────────┐
│   Family    │────<│   User      │
│   家庭       │ 1:N │   用户       │
└─────────────┘     └─────────────┘
                          │
        ┌─────────────────┼─────────────────┐
        ▼                 ▼                 ▼
   ┌────────┐        ┌────────┐        ┌────────┐
   │ Child  │        │ Parent │        │ Admin  │
   │ 孩子   │        │ 父母   │        │ 管理员 │

```
┌─────────────┐     ┌─────────────┐

## 技术选型理由

| 组件 | 选择 | 理由 |
|------|------|------|
| API | FastAPI | 高性能、自动文档、类型安全 |
| DB | PostgreSQL | 成熟可靠、JSONB支持、pgvector |
| Vector DB | Qdrant | 高性能、原生向量、成熟稳定 |
| Cache | Redis | 高速缓存、Celery后端 |
| File Storage | S3/MinIO | S3兼容、任意云 |
| Task Queue | Celery | Python生态成熟 |
